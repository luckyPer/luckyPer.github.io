题目简述:
给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。
解题思路:

  • 看到组合| 排列题, 选择用回溯算法 + 剪枝基本都能AC, 但 377. 组合总和 Ⅳ回溯算法超时了, 看了题解用的是动态规划 做的;

  • 回溯的基本模板:

    c++

  • 剪枝: 本题计算的是总和, 所以tmpSum > Sum 就可以return, 如果 数组 candidates中有重复的值, 那么在遍历的是candidates[i] == candidates[i-1] 的地方就可以containus;

  • 再执行前sort一下 数组 candidates;

  • 最好能在写代码前画出回溯树;

  • 注意在遍历的时候区分传入index / index +1 的区别: 如果需要结果是不重复的, 就传index+1;

  • 有的题还需要记忆化动态规划, 这时候就可以定义vis[len] 数组, 来去除掉已经访问的index;
