LeetCode刷题笔记: 两数之和~ 四数之和

leetcode链接:

  • 1. 两数之和
  • 15. 三数之和
  • 16. 最接近的三数之和
  • 18. 四数之和
题目简述: 在列表中求出 X 个数的和, 此和与给出的target相等.
解题思路:

  • 先排序, 复杂度O(nlogn);

  • 遍历nums列表, 定义两个左右指针 l 和 r, l 指向 i+1, r指向len-1;

  • 求和sum = nums[i] + nums[l] + nums[r], 如果sum大于target , l++; sum小于target, r–;

  • 当sum == target时, 此时求出一组答案, 如果题目要求答案中不可以重复, 那么就需要在此去重,

    c++

  • 外层nums[i]==nums[i-1]时直接continue;

复杂度为: O(N^2) + O(nlogn) = O(N^2)
